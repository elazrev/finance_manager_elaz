# PWA - התקנה כאפליקציה

האפליקציה תומכת ב-**Progressive Web App (PWA)** – ניתן להתקין אותה על אנדרואיד (ואייפון) כמו אפליקציה רגילה.

## איך להתקין באנדרואיד

1. פתח את האפליקציה ב-**Chrome** (או דפדפן אחר)
2. לחץ על **תפריט** (שלוש נקודות) → **"הוסף למסך הבית"** או **"התקן אפליקציה"**
3. אשר את ההתקנה
4. אייקון האפליקציה יופיע על מסך הבית

## איך להתקין באייפון

1. פתח את האפליקציה ב-**Safari**
2. לחץ על כפתור **"שתף"** (ריבוע עם חץ)
3. בחר **"הוסף למסך הבית"**
4. אשר את ההתקנה

## מה הוגדר

- **manifest** (`app/manifest.ts`) – שם האפליקציה, אייקונים, צבעים
- **Service Worker** (`public/sw.js`) – תמיכה בסיסית ב-offline
- **מטא-תגים** – theme-color, apple-web-app-capable

## קבצים רלוונטיים

| קובץ | תיאור |
|------|-------|
| `app/manifest.ts` | הגדרות PWA (שם, אייקונים, צבעים) |
| `public/sw.js` | Service Worker לתמיכה ב-offline |
| `public/icon-192.png` | אייקון 192×192 |
| `public/icon-512.png` | אייקון 512×512 |
| `components/pwa-register.tsx` | רישום ה-Service Worker |

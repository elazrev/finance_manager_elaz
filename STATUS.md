## 🚀 איך להתחיל

```bash
npm install
cp .env.local.example .env.local
# ערוך .env.local עם מפתחות Supabase
# הרץ את SQL מ-SUPABASE_SETUP.md
npm run dev
```

פתח http://localhost:3000
